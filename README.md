# Projeto-gama
Projeto final submetido para o HSM Coders da Gama Academy
Autor: Guilherme Rabelo

O projeto consiste em um arquivo app.js contendo o código que roda no terminal.

O programa acessa o arquivo database.JSON que contém uma lista de cursos pré-adicionados. Além de acessar essa base de dados, o programa possui diversas 
funções internas que permitem que o usuário crie, apague, edite e visualize os cursos utilizando comandos simples.
